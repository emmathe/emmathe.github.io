java.util.Random的源码阅读

介绍

java.util.Random是Java的一个实用工具类库中的一个类，提供了产生各种类型随机数的方法

结论：Random的实例是线程安全的，内部使用了原子类和CAS确保多线程下安全，但由于原子变量的更新是CAS操作，同时只有一个线程成功，造成了大量线程进行自旋重试操作，从而降低了并发性能
